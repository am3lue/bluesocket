🚀 Welcome to BlueSockets

BlueSockets is a fast and simple way to send messages and budget requests (like TZS transfers) between apps.

📖 Simple Guides

  • How it Works - The "Post Office" model.
  • Usage Guide - How to send your first message.
  • Security - How we keep your data safe.

🛠️ Key Features

  1. Never Fails: Messages are stored in a database, so they are never lost.
  2. Zero Cost: Runs for free on Vercel and Turso.
  3. Transaction History: Every message is saved so you have proof of your budget requests.
  4. Smart Sync: Only checks for messages when needed to save battery.

🚦 Quick Start

  • Open the Chat to start talking.
  • Use the Monitor to see who is online.
  • Check the Docs folder for more technical details.
